# Installation

## 1. Install PHP-Composer

### On Linux/Mac

```
$ curl -s https://getcomposer.org/installer | php -- --install-dir=bin
```

### On Windows
Please take a look at http://getcomposer.org/doc/00-intro.md#installation-windows

## 3. Install Hackathon Magento Composer

